## How to Add Screenshots

1. Take screenshots of:
   - MongoDB Compass showing telemetry data
   - Postman API testing results
   - Webhook.site alert notifications

2. Save screenshots to this folder with descriptive names

3. Reference in README.md using:
   ```markdown
   ![Description](./assets/screenshot-name.png)
   ```

## Image Guidelines

- **Format:** PNG or JPG
- **Size:** Optimize images (< 500KB each)
- **Naming:** Use lowercase with hyphens (e.g., `mongodb-telemetry-data.png`)
- **Resolution:** Clear and readable (1920x1080 or similar)
